# StumbleGuys-Strings
Coleção organizada de strings relacionadas ao Stumble Guys para fins de estudo, desenvolvimento e testes com a criação de mods via MelonLoader.

Todas as strings foram recém obtidas, pode ser utilizada para criação cheats (com nome das strings e id's dos mapas, emotes, skins, etc...) ou criação de "sevidores privados".
